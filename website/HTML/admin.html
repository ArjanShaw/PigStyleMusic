Inventory Management

Scan Records to Set Location

1

How it works:

  • Enter a location prefix (e.g., "bin 1", "shelf A")
  • Scan barcodes - each scan updates the record's location and last_seen date
  • Counter automatically increments for each scan in the current location
  • Format: "{prefix}/{counter}" (e.g., "bin 1/12")
  • Switch prefixes - counter resets to 1 for the new prefix
